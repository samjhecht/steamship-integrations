## Usage

Issues are managed via wrangler MCP tools:
- `issues_create` - Create new issues
- `issues_list` - List/filter issues
- `issues_update` - Update issue status
- `issues_search` - Search issues

## Labels

Use labels to categorize issues:
- `bug` - Something isn't working
- `feature` - New functionality
- `enhancement` - Improvement to existing functionality
- `governance` - Governance framework related
- `infrastructure` - CI/CD, tooling, configuration
- `documentation` - Documentation updates

## Priority Levels

- **critical** - Blocks all progress
- **high** - Should be addressed soon
- **medium** - Important but not urgent
- **low** - Nice to have
