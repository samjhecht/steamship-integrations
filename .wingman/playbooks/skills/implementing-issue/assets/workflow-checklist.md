# Issue Implementation Checklist

## Setup
- [ ] Read issue via MCP (issues_get with include_body: true)
- [ ] Capture working directory and branch
- [ ] Understand requirements and acceptance criteria

## Implementation (TDD)
- [ ] Write failing test (RED)
- [ ] Implement to pass test (GREEN)
- [ ] Refactor if needed (REFACTOR)
- [ ] Commit implementation

## Code Review
- [ ] Request code review
- [ ] Fix Critical issues (2 attempts max)
- [ ] Fix Important issues (2 attempts max)
- [ ] Document Minor issues

## Completion
- [ ] All tests pass
- [ ] TDD Compliance Certification provided
- [ ] Work committed
- [ ] Results reported
